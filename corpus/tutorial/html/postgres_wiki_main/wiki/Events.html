PostgreSQL Events

Most PostgreSQL-specific events are tracked on the PostgreSQL Events page. This is a listing of events at which we expect, or would like to have, a PostgreSQL presence. Please keep the events in order by starting date and follow the existing examples. Please also tag the events with the MediaWiki "PostgreSQL Events" category. If you are going to be organizing a PostgreSQL booth, please adhere to BoothPolicies. Conference for China user group is here.

OpenAM

  • Provider -- Open Identity Platform Community
  • Website -- https://github.com/OpenIdentityPlatform/OpenAM/
  • License -- open source (CDDLv1.1)
  • Interoperability level -- interoperable via generic JDBC connection
  • Verified PostgreSQL versions -- didn't actually run the program, but just checked the documentation
  • Last update (YYYY-MM-DD) -- 2018-3-18
  • Description -- Open Access Management (OpenAM) is an access management solution that includes Authentication, SSO, Authorization, Federation, Entitlements and Web Services Security. Cross Domain Single Sign On (CDSSO), SAML 2.0, OAuth 2.0 & OpenID Connect ensure that OpenAM integrates easily with legacy, custom and cloud applications without requiring any modifications. It's a developer-friendly, open-source control solution that allows you to own and protect your users digital identities.
